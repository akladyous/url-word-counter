## Backend

-   Ruby on Rails 7.0.4
-   PostgreSQL
-   Puma (web server)
-   Redis
-   FriendlyId (URL slugs)
-   WordsCounted (word counting)

## Frontend

-   Stimulus (JavaScript framework)
-   Turbo (hotwire)
-   Bootstrap 5 (styling)
-   Bootstrap Icons
-   Sass (CSS preprocessing)
-   esbuild (JavaScript bundling)

## Testing

-   Capybara (integration testing)
-   Selenium WebDriver
